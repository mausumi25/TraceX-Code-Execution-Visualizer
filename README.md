# TraceX-Code-Execution-Visualizer

AI-powered tool to visualize code execution through step-by-step video dry runs.

## Overview

TraceX helps developers understand program behavior by converting code execution into an interactive visualization. It supports multiple languages and highlights key runtime states, variable changes, and control flow.

## Features

- Visual execution flow for code snippets
- Step-by-step state inspection
- Support for backend and frontend integration
- Clear setup instructions and usage examples

## Getting Started

### Prerequisites

- Node.js 18+ (or the version required by the project)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mausumi25/TraceX-Code-Execution-Visualizer.git
   cd TraceX-Code-Execution-Visualizer
   ```
2. Install dependencies for each part of the project:
   ```bash
   cd backend
   npm install
   cd ../frontend
   npm install
   ```

### Run Locally

```bash
# From the repository root
cd backend
npm start

cd ../frontend
npm start
```

## Project Structure

- `backend/` - API server and execution engine
- `frontend/` - user interface and visualization client
- `README.md` - project overview and setup guide
